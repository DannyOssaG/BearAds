📘 Conectar Meta Business
Sigue estos pasos para obtener tu Access Token y conectar tu cuenta de Meta Ads.
PASO 1 — Crear App en Meta
Ve a developers.facebook.com/apps → Crear app → Empresa → Agrega "Marketing API"
PASO 2 — Obtener Access Token
En tu app → Herramientas → Explorador de la API Graph → Selecciona tu app → Generar token → Permisos: ads_read, ads_management

Ve a business.facebook.com → Configuración → Cuentas publicitarias → copia el ID (act_XXXXXXXXX)
